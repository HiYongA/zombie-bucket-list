## 📝 소개

“만약 세상에 좀비 사태가 터진다면 어떨까요??”
<br>
<br>
우리의 프로젝트 '**좀비가 되기 전에 해야 하는 100가지**'는 바쁜 현대인들의 일상에서 벗어나 특별한 모험을 제안합니다. <br/>
영화 '좀비100'에서 영감을 받아 시작된 이 프로젝트는 좀비로부터 해방된 시간을 활용하여 여러분만의 버킷리스트를 실현하는 기회를 제공합니다. <br/>
지루한 일상을 벗어나 새로운 경험을 시작해보세요.
